# capsule

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and deploy for production
```
gcloud app deploy
```

### Compiles and deploy for production with non-interactive
```
gcloud app deploy --quiet
```

### Open project on default browser
```
gcloud app browse
```
